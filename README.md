## Usage

### Shortcuts
* Ctrl + y : Admin Login
* Ctrl + k : Homepage
* Ctrl + n : Add new Widget (For Editing: Click on Widget)

![ScreenVideo](build/ScreenVideo%20-%20First%20Run%20and%20Edit.gif)
